# Комментарии

Компонент комментарии для битрикса. 
---
Проверяется на существование отдельный инфоблок для комментариев и его тип. Будет добавлен как тип, так и инфоблок, если отсутствуют.   

```php
	<?$APPLICATION->IncludeComponent(
	"bitrix:comments",
	"",
		Array(
			"ELEMENT_ID" => "1",
			"IBLOCK_ID" => "1",
			"IBLOCK_TYPE" => "news"
		)
	);?>
